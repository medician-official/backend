# about GIT
GIT ground rule?

- development branch에서 private branch 생성 후 기능단위로 작업.
- 작업이 끝나면 private branch → development merge & PR.
- merge 완료 후 private branch는 삭제하고 development branch에서 새로운 private branch 생성하여 작업.
